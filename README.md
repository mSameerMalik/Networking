# 🧪 Basic Network Lab using Scapy

## 🔍 Overview
This lab focuses on using **Scapy**, a powerful Python-based interactive packet manipulation tool, for basic network tasks such as packet crafting, sniffing, and analysis. The goal is to understand how raw packets are created, sent, and interpreted—key skills for any cybersecurity student or professional.

---

## 🛠️ Tools Used
- Python 3.x
- Scapy
- Linux (Tested on Kali)

---

## 🧪 Lab objectives
1. **Craft and Send Custom ICMP (Ping) Packets**
2. **Sniff and Filter Network Traffic**
3. **Perform a Basic TCP SYN Scan**
4. **Analyze Packet Layers and Headers**
